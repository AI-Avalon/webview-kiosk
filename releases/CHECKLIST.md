## インストール手順チェックリスト

### 1. デバイス所有者権限設定

工場出荷状態またはリセット直後に以下を実行：

```bash
adb shell dpm set-device-owner uk.nktnet.webviewkiosk/.WebviewKioskAdminReceiver
```

確認：
```bash
adb shell dpm get-active-admin
```

- [ ] デバイス所有者権限が設定されている
- [ ] 管理者権限確認コマンドで正しく表示される

### 2. APK インストール

```bash
adb install releases/webview-kiosk-v0.26.9-release.apk
```

- [ ] インストール成功メッセージが表示される
- [ ] デバイスでアプリが起動可能

### 3. 初期設定

アプリを起動し、以下の設定を実施：

#### セキュリティ設定
- [ ] 認証方法：パスワード を選択
- [ ] パスワード：設定済み（記録済み）
- [ ] パスワード確認：正しく入力できる

#### ウェブコンテンツ設定
- [ ] ウェブサイト URL：`https://www.e-shisetsu.e-aichi.jp/web_info.html`
- [ ] ホワイトリスト：`https://www\.e-shisetsu\.e-aichi\.jp/.*`
- [ ] ブラックリスト：空白
- [ ] URL アクセステスト：指定サイトにアクセス可能

#### タイムアウト設定
- [ ] 不活動時にリセット：600秒（10分）
- [ ] 不活動時に画面消灯：300秒（5分）
- [ ] 画面消灯時にリセット：ON

#### ロック機能設定
- [ ] Lock Task モード：ON
- [ ] 戻るボタン動作：無効
- [ ] ウェイクロック：ON

#### ホーム画面設定
- [ ] デフォルトランチャー：Webview Kiosk に設定
- [ ] 再起動後に自動起動確認

## トラブルシューティング対応

### よくある問題と解決策

| 問題 | 原因 | 解決策 |
|-----|------|--------|
| 設定画面にアクセス不可 | パスワード忘れ | デバイスリセット後、再セットアップ |
| URL フィルタリング不可 | 正規表現エラー | `\.` エスケープを確認 |
| 画面が勝手に消える | タイムアウト短すぎ | 設定確認・調整 |
| 他アプリが起動 | Lock Task 無効 | Lock Task を ON に設定 |
